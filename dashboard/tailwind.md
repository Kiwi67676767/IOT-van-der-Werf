# Tailwind installeren
1. Gebruik de standalone tailwind executable te vinden in de tailwind docs.
2. Plaats deze in de root van de folder (./)
3. Vernoem deze naar tailwind. Bijv: "tailwind.exe"

# Watcher
Start de watcher met het volgende commando:
./tailwind.exe -i ./app/static/css/src.css -o ./app/static/css/main.css --watch
